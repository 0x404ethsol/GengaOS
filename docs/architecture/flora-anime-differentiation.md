# Flora-Like Direction, Anime-Only Execution

## Intent
GengaOS follows orchestration-first UX patterns found in modern creative workflow tools, but narrows scope to anime pre-production and production.

## Similarity Targets
- Node/canvas orchestration as the primary interaction model.
- Parallel model/tool exploration for creative branching.
- Fast recomposition of workflows without leaving the canvas.

## Anime-Only Differentiation
- Identity lock enforcement before generation (`actorLockId`).
- Built-in anime shot grammar templates (dialog close-up, sakuga impact, mecha reveal).
- Style bible memory tuned for anime attributes (line weight, cel shading mode, lens language).
- Sakuga-first interpolation pipeline for in-betweens.
- Continuity copilot focused on anime-specific drift failure modes.

## Product Guardrails
- Do not generalize toward all-design categories.
- Prioritize anime pipeline depth over horizontal creative breadth.
- Every new extension capability must state anime workflow value.
